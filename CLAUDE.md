# Проект СНТ «Берёзка» — правила разработки

## 🔧 Архитектура портов (FIXED 2026-06-12, итог — PHP на :8080)

**Основной сайт = `berezka-php` (чистый PHP, MVC по структуре айтишника). Его отдаёт Apache на порту `8080`. Это единственная боевая версия.**

| Порт | Что | Путь | Статус |
|------|-----|------|--------|
| **8080** | **Apache → PHP-сайт `berezka-php`** | `D:\...\berezka-php\public` | ✅ ОСНОВНОЙ сайт — открывать ЭТОТ |
| **:3306** | MySQL / MariaDB (база `snt_berezka`) | XAMPP | ✅ Обязателен — без него сайт падает |
| **:80** | Apache (старый React-API + phpMyAdmin) | `C:\xampp\htdocs\` | ⚠️ PHP-сайтом НЕ используется |
| — | React-версия `snt-berezka — копия (2)` | этот каталог | 🗄️ Заморожена/архив, НЕ на 8080 |

### Правильные адреса:
- ✅ **http://localhost:8080** — сайт СНТ (PHP, berezka-php) **← ИСПОЛЬЗОВАТЬ ЭТОТ**
- ✅ **http://localhost/phpmyadmin** — управление базой
- ❌ Любой другой порт — неправильно

### Важно:
- На :8080 Apache отдаёт `berezka-php/public` через `<VirtualHost *:8080>` в `httpd-vhosts.conf`. В `httpd.conf` стоит `Listen 8080` — это нужно, не удалять.
- `berezka-php` — **самодостаточное PHP-приложение** (mysqli + сессии), напрямую ходит в MySQL. Папка `api/*.php` и `htdocs/api` (React-бэкенд) ему **не нужны**.
- **MySQL (MariaDB) обязателен.** Если сайт выдаёт `Fatal error ... mysqli`, значит не запущен MySQL — запустить `mysqld.exe --standalone` (или `start-snt.bat`).
- Раздел «Бюджет» (`/admin/budget`) перенесён в PHP 2026-06-12 (модель `Budget`, метод `AdminController::budget`, шаблон `admin/budget`). Таблицы `budget_items`, `budget_expenses`.
- Порт 5173 / Vite больше не используются.

### Запуск сайта (правильный способ)
```text
Двойной клик на start-snt.bat в этом каталоге:
   → стартует MySQL (MariaDB) на :3306
   → стартует Apache (PHP-сайт на :8080)
   → открывается http://localhost:8080
Остановить: stop-snt.bat (или закрыть окна Apache/MySQL).
```

**⚠️ НЕ ПУТАТЬ:**
- `berezka-php/` = ОСНОВНОЙ сайт (PHP), открывается на :8080 ← **редактируем здесь**.
- `snt-berezka — копия (2)/` = старая React-версия, заморожена, на 8080 НЕ отдаётся.

## 📋 Другие правила

### PHP и синхронизация API
- Редактировать: `...копия (2)\api\*.php`
- Копировать в: `C:\xampp\htdocs\api\`
- Команда синхронизации: скил `snt-sync`

### .bat-файлы (Windows)
- **Только CRLF** (не LF)
- UTF-8 БЕЗ BOM
- Команда `chcp 65001` в начале
- Писать через PowerShell или инструмент Write (выбирает автоматически)

### Тестирование
- Скил `snt-test` для параллельного тестирования разных ролей
- Скил `snt-debug` для отладки раздела
